# Changelog

All notable changes to the `debate` plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-24

### Added
- Initial release
- `/debate` slash command with built-in Task-based fallback
- Support for multi-provider debates via `agent-debate` Python package
- Adaptive consensus detection with configurable max rounds
- Three default personas: Architect, Pragmatist, Reliability Engineer
